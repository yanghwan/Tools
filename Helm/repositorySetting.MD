### GitHub 활용 
Helm은 기본적으로 Chart를 이용하여 Install 하여 서비스를 디플로이 한다.  
외부 저장소 github를 이용하여 간단하게  Helm chart repository를 구성 관리할수 있다.  
사전에  https://github.com/ 으로 접근하여 계정생성후 진행하면 된다.
